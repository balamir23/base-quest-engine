# Base Quest Engine

Base Quest Engine is a modular smart contract system that allows projects to create, manage, and reward on-chain user activities.

## 🚀 Features

- Create on-chain quests (tasks)
- Verify user participation (tx, mint, interaction)
- Distribute ERC20 or NFT rewards
- Admin-controlled or permissionless modes
- Gas-efficient and scalable design
- Built for Base ecosystem

## 🧠 Use Cases

- Airdrop campaigns
- User onboarding systems
- Loyalty programs
- Gamified DeFi/NFT platforms

## ⚙️ Tech Stack

- Solidity ^0.8.x
- OpenZeppelin
- Base (L2)
